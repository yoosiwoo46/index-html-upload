# 🚗 代驾司机模拟器 V1.3

> 急头白脸就是干 —— 一款沉浸式代驾司机模拟经营游戏

## 📖 游戏简介

《代驾司机模拟器》是一款以代驾司机为视角的模拟经营网页游戏。你将从一名普通代驾司机起步，在多个代驾平台上接单赚钱，管理财务、提升能力、经营人生，最终目标是成为"代驾之王"。

## 🎮 基本玩法

### 1. 角色创建

游戏开始时，你需要创建自己的角色：
- **姓名**：输入你的司机名
- **性别**：选择男/女
- **年龄**：初始 28 岁
- **驾龄**：初始 3 年
- **城市**：选择你所在的工作城市

### 2. 核心循环

游戏以**时间推进**为核心机制，每次循环推进 6 分钟游戏时间。每天的流程大致为：

```
起床 → 上线接单 → 完成订单赚钱 → 管理经营 → 收工睡觉 → 进入下一天
```

### 3. 接单方式

| 方式 | 说明 |
|------|------|
| **平台派单** | 加入平台后上线等待，系统自动派单（滴滴1分钟/单、高德3分钟/单、e代驾5分钟/单） |
| **抢单大厅** | 多个订单同时展示，手动抢单，手速决定一切 |
| **线下报单** | 不依赖平台，线下寻找客户，自由定价，收入更高但更不稳定 |
| **回头客** | 服务好的客户可能成为回头客，7天后主动联系你 |

### 4. 订单类型

| 类型 | 占比 | 说明 |
|------|------|------|
| **实时单** | 45% | 按距离计费，最常见 |
| **一口价** | 25% | 固定价格，简单省心 |
| **取送车** | 10% | 需要先考取取送车认证 |
| **包时** | 7% | 按时长计费 |
| **代泊车** | 5% | 按时长计费 |
| **万能司机** | 3% | 按时长计费 |
| **跨城单** | 5% | 远距离高收入订单 |

### 5. 多平台经营

游戏支持三大代驾平台，各有特点：

- **滴滴（DD）**：派单最快（1分钟/单），竞争激烈
- **高德（GD）**：派单间隔适中（3分钟/单）
- **e代驾（E）**：派单间隔较长（5分钟/单）

> 💡 可以同时加入多个平台，多平台同时上线接单效率更高！

## 📊 角色属性

| 属性 | 说明 | 提升方式 |
|------|------|----------|
| **驾驶能力** | 影响订单完成质量 | 完成订单、购买培训 |
| **话术能力** | 影响小费和客户满意度 | 完成订单、购买培训 |
| **豪车驾驶** | 影响能否接豪车订单 | 完成豪车单、购买培训 |
| **代驾分** | 平台评分，过低会被封禁 | 保持良好服务，避免事故 |
| **服务分** | 影响接单优先级 | 完成月度任务、避免投诉 |

## 🛒 商店与装备

### 代步工具
- **折叠电动车**：代驾必备，没有代步车无法接单！可全款或分期购买

### 平台装备
- **平台马甲**：提升客户信任度
- **保温杯 / 饭盒**：减少餐饮支出
- **充电宝**：保持手机电量
- **雨衣 / 手套 / 护膝**：应对恶劣天气

### 平台增值服务
- **保上保**：15天内车损免赔免扣代驾分
- **优质订单优先享**：前10笔派单均为50km+优质单
- **超值免佣包**：8天内订单收入 ×1.2

## 💰 经营管理

### 财务系统
- **房租**：每月 3000 元（有房贷且未还清时免房租）
- **房贷**：结婚后可能需要还房贷（3~8年期）
- **贷款**：资金不足时可申请贷款（消耗征信）
- **存款利息**：余额为正时每月获得 0.5% 利息
- **个税**：月收入超过 800 元需缴纳个人所得税（房租/房贷可抵扣）

### 理财系统
- **银行存款**：稳定利息收入
- **基金股票**：高风险高收益
- **乐透人生**：彩票，小概率大回报

### 包工头模式
当存款超过 5 万元时，可以晋升为**包工头**：
- 招募其他司机为你工作
- 从手下司机的订单中抽取分成
- 最多管理 4 名司机

## 👨‍👩‍👧 家庭系统

- **结婚**：每月有概率遇到合适的对象步入婚姻
- **子女**：结婚后可能生育子女，需要承担学费等开支
- **配偶事件**：偶发的家庭开支（纪念日礼物等）

## 🌦️ 天气与突发事件

| 事件 | 效果 |
|------|------|
| **下雨/下雪** | 订单量增加，但路滑有受伤风险（5%概率） |
| **受伤** | 休息 7 天，保险赔付 1000 元 |
| **交通事故** | 需要赔偿车损，代驾分 -6，服务分 -10 |
| **生病** | 睡眠不足导致，需要花费看病 |
| **客服外呼** | 下雨天平台可能打电话催你上线 |

## 📚 培训系统

### 线上培训
- **每周答题**：回答驾驶/交规题目，答对获得奖励
- 涵盖交通信号、安全驾驶、高速公路规则等真实交规知识

### 线下培训
- 每月需参加一次线下培训
- ⚠️ 未参加线下培训的月份将**无法接单**！

## 🏆 称号系统

游戏内置多个隐藏称号，在月度结算时根据表现解锁：

| 称号 | 解锁条件 |
|------|----------|
| 「有单就是干」 | 月度平台派单超过 150 单 |
| 「传奇报单王」 | 月度线下报单超过 100 单 |
| 「F1预备役」 | 月度车损赔偿超过 10000 元 |
| 「超绝包工头」 | 月度包工头抽成收入超过 10000 元 |
| 「这把高端局」 | 月度豪车订单超过 80 单 |
| 「常在河边走 哪有不湿鞋」 | 月度被平台扣分 |

## 🏁 游戏结局

游戏有多种结局，取决于你的最终资产：

| 结局 | 条件 |
|------|------|
| 🥇 **代驾之王** | 最终金钱 > 10 万元 |
| 🥈 **金牌司机** | 最终金钱 > 5 万元 |
| 🚗 **普通司机** | 最终金钱 > 2 万元 |
| 😅 **勉强糊口** | 最终金钱 ≤ 2 万元 |
| 💀 **破产结局** | 资金耗尽 |
| ⚠️ **代驾分清零** | 代驾分降为 0 |
| 🚫 **失信人** | 征信耗尽 |

## ⚠️ 游戏注意事项

1. **每天最多在线 14 小时**，超时强制下线
2. **睡眠不足 5 小时**会被强制休息
3. **代驾分降至 0** 会被平台封禁
4. **征信耗尽**且资金为负则游戏结束（失信人结局）
5. **代步车使用超过 2 年**会损坏报废，需重新购买
6. 记得及时**保存游戏**，存档存储在浏览器本地
